# VoteSense AI: Your Intelligent Voting Decision Companion 🇮🇳

**VoteSense AI** is a premium, cinematic web application designed to empower voters (especially first-timers) by bridging the gap between democratic complexity and individual action. Built with modern web technologies, it provides a personalized, AI-guided journey from registration to the polling booth.

🚀 **Live Demo:** [https://ai-voting-decision-companion.vercel.app](https://ai-voting-decision-companion.vercel.app)

---

## 🌟 Key Features

### 1. 🧭 Election Guide Mode (NEW)
A comprehensive, 5-step educational roadmap that helps users understand the legal and logistical foundation of their voting rights:
- **Registration**: Understanding the foundation of voting rights.
- **Verification**: Ensuring your name is on the official electoral roll.
- **Booth Discovery**: Finding your exact assigned polling location.
- **ID Preparation**: Mastering mandatory identification requirements.
- **Voting Day**: Navigating the final steps at the booth.

### 2. 🧠 Journey-Aware AI Assistant
An intelligent companion that contextually explains your current status in the election process. It identifies critical blockers and provides immediate, actionable solutions.

### 3. 🎮 Voting Day Simulator
A high-fidelity, interactive simulator that walks users through the exact experience of visiting a polling booth, performing identity verification, and using an Electronic Voting Machine (EVM).

### 4. 📊 Personal Readiness Dashboard
A centralized control center showing your "Readiness Score," estimated time remaining to complete tasks, and a dynamic vertical roadmap.

## 🛠️ Technology Stack

- **Frontend**: React 19 + Vite
- **State Management**: Zustand (with LocalStorage persistence)
- **Animations**: GSAP (ScrollTrigger, ScrollTo) + Framer Motion
- **Styling**: Styled-Components + Vanilla CSS (Glassmorphism design system)
- **Deployment**: Vercel

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Parthvi1707/AI-Voting-Decision-Companion.git
   ```
2. Navigate to the project directory:
   ```bash
   cd AI-Voting-Decision-Companion
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## 📸 Design Philosophy
The application follows a **Premium Dark Aesthetic** with "Liquid Glass" effects:
- **Glassmorphism**: Layered cards with blur and transparency.
- **Dynamic Glows**: Saffron, White, and Green (Tricolour) ambient orbs.
- **Cinematic Transitions**: Scroll-driven storytelling using GSAP.
